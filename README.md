# jiractl

CLI tool for interacting with Jira.

## Installation

### From Source

```bash
go install github.com/e/jiractl/cmd/jiractl@latest
```

### Build Locally

```bash
make build
```

## Usage

```bash
jiractl --help
```

## Development

```bash
# Build
make build

# Run tests
make test

# Run linter
make lint

# Run locally
make run ARGS="--help"
```
